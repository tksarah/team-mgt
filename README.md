# Team Progress Manager β1.2

Team Progress Manager は、チームで共有しながら活動・進捗を管理するための Windows デスクトップアプリです。Tauri を使ったポータブル構成で、一覧表ベースの扱いやすい UI と、共有フォルダ運用を意識した編集ロック機能を備えています。


<img width="1455" height="1314" alt="image" src="https://github.com/user-attachments/assets/52251960-4838-444b-9c29-158a0e50dde5" />

## 概要

- タスク名、カテゴリ、担当者、ステータス、優先度、開始日、期日、依存タスクを一元管理できます。
- 複数担当者の設定、タスク間の前後関係、メモの記録に対応しています。
- カテゴリ、担当者、ステータス、優先度は設定画面から編集できます。
- Excel のエクスポートとインポートに対応し、バックアップや配布、一覧共有に使えます。
- 共有フォルダ上の SQLite を指定して、複数人で同じデータを参照できます。
- 編集中は `.lock` ファイルを作成し、同時更新の衝突を避けます。

## 想定利用シーン

- 小規模から中規模チームの進捗管理
- 定例会議用の進捗一覧の確認
- Excel ベース運用からデスクトップアプリへの移行
- 共有フォルダを使ったシンプルな社内タスク管理

## 動作環境

- Windows
- ポータブル実行対応
- ローカルまたは共有フォルダ上の SQLite データファイル
